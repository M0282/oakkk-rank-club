# 검증 항목

패키지 생성 시 다음 항목을 자동 검사합니다.

- 최근 전적 수집 개수 상수: 20
- 솔로랭크 `queueId = 420`만 듀오 집계
- 같은 경기·같은 팀인 등록 플레이어만 조합 생성
- 2경기 1승 1패 → 승률 50%
- 예측 판정 시간: 8시간
- 보상: `0/±20/±40/±60 → 5/10/15/20`
- 허용되지 않은 점수 변동폭에는 보상 0
- Riot 경기 상세 → 참가자 캐시 변환
- SQL에서 `score_delta` 사용
- 루트 SQL과 Supabase 마이그레이션 파일 일치
- `package-lock.json`에 내부 전용 npm 주소가 없음
- 전체 JavaScript 구문 검사
- Vite 프로덕션 빌드

실제 Riot API Key와 실제 Supabase 프로젝트에 대한 네트워크 통합 검사는 사용자의 배포 환경에서 최종 확인해야 합니다.
