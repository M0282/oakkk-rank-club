# 검증 기록

이 패키지는 다음 검사를 통과했습니다.

## 자동 검사

1. `npm test`
   - 같은 `matchId` + 같은 `teamId` 조합만 집계되는지 확인
   - 자유랭크 `queueId = 440`이 제외되는지 확인
   - 2경기 1승 1패가 50%로 계산되는지 확인
   - 예측 방향과 오프셋별 보상 5/10/15/20 계산 확인
   - Riot Match-V5 응답을 경기/참가자 캐시 형식으로 변환하는 로직 확인

2. `npm run build`
   - Vite 프로덕션 빌드 성공
   - React/JSX 전체 변환 성공

3. 서버 파일 구문 검사
   - `api/**/*.js`
   - `lib/**/*.js`
   - 모든 파일 `node --check` 통과

4. 의존성 잠금 파일 검사
   - `package-lock.json`과 `package.json` 일치 확인
   - 오프라인 `npm ci` 성공
   - OpenAI 내부 전용 npm 레지스트리 주소가 남아 있지 않음

## 외부 환경에서 최종 확인해야 하는 항목

실제 Riot API 키와 사용자의 Supabase 프로젝트에는 접근하지 않았으므로 다음은 배포 후 확인해야 합니다.

- Supabase에서 `migration_2026_07_complete.sql`이 오류 없이 실행되는지
- Riot API 키가 만료되지 않았는지
- 등록 플레이어의 최근 20경기가 순차적으로 채워지는지
- 실제 같은 팀 기록이 듀오 통계 화면에 나타나는지

마이그레이션을 먼저 실행하고 코드를 배포하도록 적용 순서를 구성했습니다.
